## 개요

KT / LG U+ 등 통신비를 **카드결제**로 정산할 때 사용하는 전표입니다.
카드결제이므로 세금코드 **V3** 를 반드시 입력해야 합니다.

> **세금코드 V3란?**
> 법인카드(신용카드)로 결제 시 매입 부가세를 공제받기 위해 입력하는 세금코드입니다.

---

## 전표 항목 구성

| Itm | PK | 계정번호 | 내역 | 비고 |
|-----|-----|---------|------|------|
| 001 | 31 | 공급업체 계정 | 미지급금 (대변) | 카드번호 입력 |
| 002 | 40 | 11161100 | 선급부가가치세 | LG U+ 부가세 |
| 003 | 40 | 50034300 | (판)PC,인터넷통신료 | LG U+ 공급가액 |
| 004 | 40 | 50034100 | (판)전화사용료 | KT 공급가액 |
| 005 | 40 | 11161100 | 선급부가가치세 | KT 부가세 |

---

## STEP 1. 전표 헤더 입력

- **T-CODE** : F-63 (송장입시서정-일반)
- **등록일 = 전기일 = 기간(月)** 동일하게 기재
- **참조** : 부서명 / 작성자명 (예: 천안지점/홍길동)
- **전표헤더텍스트** : 입력 없음 (텍스트 없음 체크)
- **통화** : KRW

---

## STEP 2. 항목 001 입력 (대변 / PK 31)

<div style="background:#f8f9fa; border-left:4px solid #5b8db8; padding:12px 16px; border-radius:4px; margin:12px 0;">

- **공급업체** : 카드사 공급업체 코드 입력
- **G/L 계정** : 21031600
- **금액** : 청구서 합계금액 (마이너스 `-` 로 입력)
- **세금코드** : **V3**
- **사업장** : BP01
- **지급 기산일** : 청구서 납기일
- **지급 방법** : T
- **텍스트** : `[월] [지점명] 비용 정산` (예: 6월 천안지점 비용 정산)

</div>

---

## STEP 3. 항목 002 입력 (차변 / PK 40) — LG U+ 부가세

<div style="background:#f8f9fa; border-left:4px solid #5b8db8; padding:12px 16px; border-radius:4px; margin:12px 0;">

- **G/L 계정** : 11161100 (선급부가가치세)
- **금액** : LG U+ 부가세액
- **기준 금액** : LG U+ 공급가액
- **세금코드** : **V3**
- **사업장** : BP01
- **지급 기산일** : 말일 (예: 2026.06.30)
- **텍스트** : `lg_[지점명]_[월]` (예: lg_천안지점_6월)

</div>

---

## STEP 4. 항목 003 입력 (차변 / PK 40) — LG U+ 공급가액

<div style="background:#f8f9fa; border-left:4px solid #5b8db8; padding:12px 16px; border-radius:4px; margin:12px 0;">

- **G/L 계정** : 50034300 (판)PC,인터넷통신료
- **금액** : LG U+ 공급가액
- **세금코드** : **V3**
- **사업장** : BP01
- **코스트센터** : 해당 부서 코스트센터 입력
- **텍스트** : `LG U+ 정산`

</div>

---

## STEP 5. 항목 004 입력 (차변 / PK 40) — KT 공급가액

<div style="background:#f8f9fa; border-left:4px solid #5b8db8; padding:12px 16px; border-radius:4px; margin:12px 0;">

- **G/L 계정** : 50034100 (판)전화사용료
- **금액** : KT 공급가액
- **세금코드** : **V3**
- **사업장** : BP01
- **코스트센터** : 해당 부서 코스트센터 입력
- **텍스트** : `KT 정산`

</div>

---

## STEP 6. 항목 005 입력 (차변 / PK 40) — KT 부가세

<div style="background:#f8f9fa; border-left:4px solid #5b8db8; padding:12px 16px; border-radius:4px; margin:12px 0;">

- **G/L 계정** : 11161100 (선급부가가치세)
- **금액** : KT 부가세액
- **기준 금액** : KT 공급가액
- **세금코드** : **V3**
- **사업장** : BP01
- **지급 기산일** : 말일 (예: 2026.06.30)

---

## STEP 7. 개요 확인 후 저장

- 차변(D) 합계 = 대변(C) 합계 확인
- 항목 수 및 세금코드 V3 입력 여부 확인
- 이상 없으면 전표 → **완료로 저장**
